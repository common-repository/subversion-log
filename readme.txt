== Description ==
This plugin displays a list of the most recent commits to an external subversion repository, along with the commit comment.

== Installation ==
1. Upload the `subversion-log` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the plugin through `extension admin menu`
4. Add a widget
